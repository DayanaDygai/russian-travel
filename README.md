# Проектная работа
## Russian-Travel
___

Этот проект был выполнен в ходе обучения на платформе «Яндекс практикум» в 3 спринте

В ходе работы использовались:
- Флексбокс-верстка;
- Гриды;
- Методология БЭМ;
- Адаптивная верстка;
***
